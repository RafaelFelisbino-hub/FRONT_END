Faça uma página web para tratar das autuações por excesso de velocidade. Precisa ter um input onde o usuário digite a velocidade máxima permitida pela via. Precisa ter um input onde o usuário digite a velocidade do carro. Criar uma div para exibir o resultado. Dependendo da velocidade exibir:

Até 7 km/h acima da permitida pela via informar: Advertência - Acima de 7 km/h.
Até 20% informar: Autuado. Infração Grave
Entre 20% e 50% informar. Autuado. Infração gravíssima.
Acima de 50% informar. Autuado.Infração gravíssima. Penalidade de suspensão do direito de dirigir.
